What is a dynamic copyright year? while it is pretty self explainatory, but I'll go an extra mile and explain it nonetheless. It is that year we have in our websites footer. You know that nobody cares about. Yea, that!

This post describes a short and easy way to include a dynamic copyright year inside the footer of a page. Infact I used this same trick on this website. Afterall who likes messing with the static content... right?... right? or is it just me?

  • Start by adding a target element inside the footer. In our case we'll be targetting #year id
  • Create a Javascript Date() object and assign it to a variable today
  • Then using jquery $('#year') method target the element inside footer
  • Now manupilcate the inner HTML using .html method and giving it a value of today variable we created earlier

{% endhighlight %} {% highlight javascript %} (function() { $(document).ready(function() { var today = new Date(); $('#year').html(today.getFullYear()); }); })(); {% endhighlight %}

There you have it, in few easy steps. Your copyright year is now fully dynamic and it will automatically update itself everytime the current year changes. Very simple and easy.
